# Dash0 Agent0 — stack integration and MCP

## Core idea

Agent0 connects to the tools teams already use — GitHub for code and commits, Linear for issue tracking, and any Model Context Protocol (MCP) server — reading from where work already lives instead of requiring data re-entry. MCP is the open extension point: it lets Agent0 reach arbitrary external context and systems through a standard protocol rather than bespoke one-off integrations. Meeting the team in its existing tools is what makes the agent's output land in the real workflow — a PR in GitHub, an update in Linear.
