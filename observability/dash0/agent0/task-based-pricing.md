# Dash0 Agent0 — task-based pricing

## Core idea

Agent0 is metered with task-based credits rather than per-interaction fees, where a task represents a meaningful action such as an investigation, an asset created, or a pull request opened. Pricing by outcome means cost tracks the work delivered, not the number of messages exchanged during it. This sits alongside Dash0's volume-based platform pricing as the separate meter for the agent's autonomous work.
