# Dash0 Agent0 — root cause and code

## Core idea

Agent0 closes the gap between telemetry and source by reading your codebase, locating the root cause within a trace, and pointing directly at the relevant code — all in one interface without context switching. Connecting the failing span to the line of code behind it is what makes its analysis actionable rather than advisory. This trace-to-code link is also the prerequisite for the next step: generating a concrete fix.
