# Dash0 alerting — fundamentals

## Core idea

Alerting lets you define checks over logs, metrics, and traces using real PromQL or a query builder, then route notifications through email, Slack, and other integrations. Using genuine PromQL — not a lookalike — means existing Prometheus rules and 400+ pre-built alert templates from open-source knowledge bases work without translation, so teams start from a large library instead of a blank page. Like dashboards, alert rules are declarative definitions, which is what makes them manageable as code and reusable across environments.
