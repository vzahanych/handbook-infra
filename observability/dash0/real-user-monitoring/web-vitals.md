# Dash0 website monitoring — web vitals

## Core idea

Web vitals are the standard page-performance metrics — load and render timings and interaction responsiveness — measured from real user sessions in the browser. They quantify perceived speed, the experience a user actually has, rather than server-side response time alone. Measured from real traffic, they reflect the true distribution across devices and networks instead of a single lab number.
