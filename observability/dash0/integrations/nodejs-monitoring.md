# Dash0 integrations — Node.js monitoring

## Core idea

Node.js monitoring is a language specialization that ships tuned OpenTelemetry instrumentation for the Node runtime plus curated dashboards and alerts. It surfaces the signals that matter for Node specifically — event-loop lag, garbage collection, HTTP and framework spans — on top of the standard traces, metrics, and logs. Because it is OTel under the hood, it integrates without a proprietary agent and the data correlates with the rest of the stack.
