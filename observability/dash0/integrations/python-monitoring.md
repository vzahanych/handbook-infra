# Dash0 integrations — Python monitoring

## Core idea

Python monitoring tailors OpenTelemetry instrumentation to Python applications and the frameworks they commonly run on, such as Django, Flask, and FastAPI. It captures request spans, database and HTTP client calls, and runtime metrics, then ships them as OTLP to Dash0 with curated dashboards and alerts on top. The value is the same pattern repeated per language: standard instrumentation, runtime-aware defaults, and one correlated store.
