# Dash0 integrations — Google Cloud monitoring

## Core idea

Google Cloud monitoring integrates natively with GCP to pull in resource metrics and logs from managed services, extending observability to cloud infrastructure you do not instrument yourself. It bridges Google Cloud's own telemetry into Dash0's unified store so platform signals sit beside your application traces and metrics. That common home is what lets a managed-service problem be correlated with the application behavior it affects.
