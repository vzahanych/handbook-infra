# Dash0 dashboards — Perses

## Core idea

Perses is the open-source, CNCF-aligned dashboard project that Dash0 adopts as its dashboarding layer instead of a proprietary format. Building on an open standard means dashboard definitions follow a documented schema that other tools can also read and write. This is the dashboard analogue of Dash0's broader bet on open standards — Perses for visualization the way OTLP is for ingest and PromQL is for queries.
