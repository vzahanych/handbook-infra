# Dash0 dashboards — import/export portability

## Core idea

Full import/export compatibility means a dashboard is a portable definition you own, not a record trapped in the vendor's UI. You can export it, store it in Git, move it between accounts or tools, and import community-authored dashboards in the same format. Portability is the concrete mechanism behind "no lock-in" for visualization, and it is what lets dashboards live in the observability-as-code pipeline alongside the application they describe.
