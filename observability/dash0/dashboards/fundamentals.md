# Dash0 dashboards — fundamentals

## Core idea

Dashboards in Dash0 are built on Perses, the open-source dashboard standard, to visualize metrics, logs, and traces with variables and pre-made templates. Choosing Perses keeps dashboards fully import/export compatible, so definitions are portable and free of vendor lock-in — you can version them, move them, and reuse community templates. Because dashboards are plain definitions rather than UI-only artifacts, they slot directly into the observability-as-code workflow and ship alongside application releases.
